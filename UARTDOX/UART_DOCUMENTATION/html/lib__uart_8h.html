void initUSART ( void )

Initialise la communication UART. Configure le baud rate, le mode asynchrone double vitesse, active l'émission et la réception, et définit le format des données (8 bits, 1 stop bit).

Note
Cette fonction doit être appelée avant toute autre opération UART.
Warning
Assurez-vous que le microcontrôleur est configuré pour utiliser la fréquence d'horloge correcte.
